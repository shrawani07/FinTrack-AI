💰 FinTrack AI

An AI-powered personal finance management platform built using Next.js, TypeScript, MongoDB, and OpenAI APIs.
FinTrack AI helps users track expenses, analyze spending habits, and get smart AI-driven financial insights through an interactive dashboard.

⚙️ Installation
Clone the repository:

git clone: https://github.com/shrawani07/FinTrack-AI

Move to the project folder:

cd fintrack-ai

Install dependencies:

npm install
🔑 Environment Variables

Create a .env.local file and add:

MONGODB_URI=your_mongodb_uri
OPENAI_API_KEY=your_openai_api_key
NEXTAUTH_SECRET=your_secret_key

▶️ Run Locally
npm run dev

Open:
http://localhost:3000
